Listar conteúdo das pastas<br>
```
ls
```

Abrir arquivo (& no final não prende o terminal)<br>
```
gedit arquivo.txt &
```

Reiniciar em 1 minuto<br>
```
shutdown -r 1
```

Cancelar operação de shutdown<br>
```
shutdown -c
```

Reiniciar instantaneamente<br>
```
reboot
```

Desligar<br>
```
poweroff
```

Ver o tamanho das pastas<br>
```
du -sh Docker
```
